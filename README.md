# 🏋️ 국민체력100 데이터 기반 근감소증 예측 서비스
클라우드 인프라(AWS)와 MLOps 파이프라인을 활용한 End-to-End AI 프로젝트입니다.

## 📅 프로젝트 로드맵
- **1주차**: GitHub 협업 환경 구축 및 Docker 컨테이너화
- **2주차**: AWS 인프라(EC2, S3) 설정 및 모델 학습 파이프라인 연결
- **3주차**: FastAPI를 활용한 모델 서빙 API 개발 및 CI/CD 구축
- **4주차**: 클라우드 배포 및 API 문서화(Swagger)

## 🛠 Tech Stack
- **Language**: Python 3.9
- **Library**: Pandas, Scikit-learn
- **Infrastructure**: AWS (EC2, S3), Docker
- **CI/CD**: GitHub Actions
- **API**: FastAPI
